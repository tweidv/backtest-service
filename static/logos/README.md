# Logo Files

Place the following logo files in this directory:

- `dome-logo.png` or `dome-logo.svg` - Dome API logo
- `polymarket-logo.png` or `polymarket-logo.svg` - Polymarket logo  
- `kalshi-logo.png` or `kalshi-logo.svg` - Kalshi logo

**Recommended:**
- Use SVG format for crisp scaling
- Keep logos simple and recognizable at small sizes (24-32px height)
- Transparent backgrounds work best
- If using PNG, use 2x resolution for retina displays

**Where to get logos:**
- Dome: https://domeapi.io (check their website or press kit)
- Polymarket: https://polymarket.com (check their website or brand assets)
- Kalshi: https://kalshi.com (check their website or brand assets)
